# Python-GUI-Applications-Tkinter-Kivy-
Desktop GUI applications built using Python libraries such as Tkinter and Kivy.
This repository contains graphical user interface (GUI) applications built using Python. These projects demonstrate how Python can be used to build desktop software.

## Projects Included

### 1. Digital Clock
A real-time digital clock showing hours, minutes, seconds, and date.

### 2. Internet Speed Test App
A GUI application that measures download and upload speed.

### 3. Calendar App
A desktop application that displays a complete calendar for any selected year.

### 4. Desktop Notifier
A program that sends desktop notifications at regular intervals.

### 5. To Do List Management App
A simple task management system to organize daily tasks.

### 6. Kivy App
An experimental GUI application built using the Kivy framework.

## Technologies Used
- Python
- Tkinter
- Kivy
- GUI design concepts

## Purpose
These projects help understand how to create interactive desktop applications using Python.
